# ds_shravi
Analyze how trading behavior (profitability, risk, volume, leverage) aligns or diverges from  overall market sentiment (fear vs greed). Identify hidden trends or signals that could influence  smarter trading strategies. 
---
## Dataset Information
Due to GitHub file size limits, the full historical dataset is not included in this repository.

Datasets used in this project:
- historical_data.csv (excluded due to size)
- fear_greed_index.csv (included)

The datasets can be accessed from:
Historical Data : https://drive.google.com/file/d/1IAfLZwu6rJzyWKgBToqwSmmVYU6VbjVs/view?usp=sharing
